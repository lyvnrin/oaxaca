Module: CustomerLogin

Handles customer authentication by allowing users to select a table number (1-20) and creates a new customer session in the backend. Upon successful authentication, navigates to the menu page with customer context.

(async, inner) handleContinue()

Handles form submission to create a new customer session Sends a POST request to create a new customer with the selected table number. On success: - Stores session flag in sessionStorage - Navigates to menu page with customer ID and table ID On failure: - Displays error message from backend or network error
